字节大小

1 byte = 8 bit
1 k = 1024 byte

进制组成与换算

二进制

二进制(0b开头,由0,1组成)
eg : (10110110) = 2^7*1 + 2^6*0 + 2^5*1 + 2^4*1 + 2^3*0 + 2^2*1 + 2^1*1 + 2^0*0 = 182(十进制结果)

十进制

十进制(由0…..9组成)
eg : (182)

八进制

八进制(0开头,由0…7组成)
从右开始,每三位一个组合,最左边不够补0,然后分别算出十进制的值进行组合得到八进制的值.
eg : (0266) 010|110|110 = 2^2*0 + 2^1*1 + 2^0*0 | 2^2*1 + 2^1*1 + 2^0*0 | 2^2*1 + 2^1*1 + 2^0*0 = 266(八进制结果) = 2*8^2 + 6*8^1 + 6*8^0 = 182(十进制结果)

十六进制

十六进制(0x开头,由0….9,a…f组成)
同理与八进制,每四位一个组合.
eg : (0xB6) 1011|0110 = 2^3*1 + 2^2*0 + 2^1*1 + 2^0*1 | 2^3*0 + 2^2*1 + 2^1*1 + 2^0*0 = B6(十六进制结果) = 11*16^1 + 6*16^0 = 182(十进制结果)

原码,反码,补码

原码

原码 : 正数的原码最高位是0.负数的最高位是1,其他的是数值位.
eg : 值   符号位   数值位
      +7      0       0000111
      -7       1       0000111

反码

反码 : 正数的反码和原码相同.负数的反码与原码符号位相同,数值位取反(就是1变0,0变1).
eg : 值   符号位   数值位
      +7      0       0000111
      -7       1       1111000

补码

补码 : 正数的补码与原码相同.负数的补码是在反码的基础上加1.
eg : 值   符号位   数值位
      +7      0       0000111
      -7       1       1111001

Java初学小问题

发表于 2018-12-04 | 更新于 2018-12-05 | 分类于 Java

CMD编译格式

javac + 文件名+ 扩展名 eg : javac HelloWorld.java (编译)
java + 类名(不加扩展名) eg : java HelloWorld (执行)

Java标识符及命名规则(见名知意)

标识符组成

数字、字母、$、_
注意 : Java关键字不可以作为标识符，(go to,const)为旧版本保留字段亦不可作为标识符使用。

包的命名

文件夹 单极 : xxx
多级 : xxx.xxx.xxx
(全部字母小写)

类或接口

一个单词 : 单词的首字母必须大写 eg : Student,Dog
多个单词 : 每个单词的首字母必须大写 eg : HelloWorld,StudentName

方法或变量

一个单词 : 单词的首字母小写 eg : age,student
多个单词 : 首字母小写,从第二个字母开始首字母大写 eg : studentAge,studentAllNames

常量

一个单词 : 全部大写 eg : PI
多个单词 : 每个字母都大写,用_隔开 eg : STUDENT_MAX_AGE
