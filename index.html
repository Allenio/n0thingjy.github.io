内存介绍

栈内存:存放的是局部变量.(局部变量:在方法定义中或者方法声明上的变量都称为局部变量)
堆内存:存放的是所有new出来的东西.

堆内存的特点

A:每一个new出来的东西都有地址值.
B:每个变量都有默认值.
  byte、short、int、long 默认值都是 0
  float、double 默认值都是 0.0
  char 默认值 '\u0000'
  boolean 默认值 false
  引用类型 默认值 null
C:使用完毕就变成了垃圾,但是并没有立即回收.会在垃圾回收器空闲的时候回收.

栈内存的特点

栈内存的数据用完就释放掉.

选择结构(switch)

switch语句结构 :

格式解释 :

switch:表示这是switch结构.
表达式:这个地方的取值是有限定的.
eg:byte、short、int、char,
   JDK5以后可以是枚举,
   JDK7以后可以是String.
case:后面跟的是要和**表达式**进行比较的值.
语句体:要执行的代码.
break:表示中断,结束的意思,可以控制switch的中断.
default:当所有值都和表达式不匹配的时候,就执行default控制的语句,相当于if语句中的else.

switch语句的注意事项 :

A:case后面只能是常量,不能是变量.而且,多个case后面的值不能相同.
B:default可以省略,但不建议,因为它的作用是对不正确的情况给出提示.
  特殊情况省略:case就可以把值固定 eg:选择题(A、B、C、D选项)
C:break可以省略,但是结果可能不是想要的.
D:default可以在任意位置,但建议在最后.
E:switch语句的结束条件:
  1.遇到break就结束.
  2.执行到末尾就结束.

逻辑运算符

运算符 含义 用法 结合方向
&&、& 逻辑与 op1&&op2 从左到右
|| 逻辑或 op1||op2 从左到右
! 逻辑非 !op 从右到左

位运算符

运算符 运算 举例
<< 左移 3 << 2 = 12 -> 3 * 2 * 2 = 12
>> 右移 3 >> 1 = 1 -> 3 / 2 = 1
>>> 无符号左移 3>>>1 = 1 -> 3 / 2 = 1
& 与运算 6 & 3 = 2
| 或运算 6 | 3 = 7
^ 异或运算 6 ^ 3 = 5
~ 反码 ~6 = 7

要做位运算,首先要把数据转换为二进制. 注 : 二进制补码运算

<<位运算符

<< : 左移(左边最高位丢弃,右边补齐0)

eg : 3 << 2 = 3 * 2 * 2 = 12

计算出3的二进制补码:
左移2位 00000000 00000000 00000000 00000011
最高位(00)舍弃 00000000 00000000 00000000 00001100

结论 : 把<<左边的数据*(乘以) 2的移动次幂

>>位运算符

>> : 右移 最高位是0,左边补齐0;最高位是1,左边补齐1

eg : 24 >> 2 = 24 / 2 ^ 2 = 6

计算出24的二进制补码:
右移2位 00000000 00000000 00000000 00011000
最高位补齐00 00000000 00000000 00000000 00000110

eg : -24 >> 2 = -24 / 2 ^ 2 = -6

计算出-24的二进制补码:
右移2位 11111111 11111111 11111111 11101000
最高位补齐11 11111111 11111111 11111111 11111010

结论 : 把>>左边的数据(除以) 2的移动次幂

>>>无符号右移

>>> : 无论最高位是0还是1,左边补齐0

eg : 24 >>> 2 = 6

计算出24的二进制补码:
右移2位 00000000 00000000 00000000 00011000
最高位补齐00 00000000 00000000 00000000 00000110

eg : -24 >>> 2 = 1073741818

计算出-24的二进制补码:
右移2位 11111111 11111111 11111111 11101000
最高位补齐00 00111111 11111111 11111111 11111010

&位运算(有0则0)

eg : 3 & 4 = 0

数值 转换为二进制
3 00000000 00000000 00000000 00000011
4 00000000 00000000 00000000 00000100
&位运算后结果为0 00000000 00000000 00000000 00000000

|位运算(有1则1)

eg : 3 | 4 = 7

数值 转换为二进制
3 00000000 00000000 00000000 00000011
4 00000000 00000000 00000000 00000100
|位运算后结果为7 00000000 00000000 00000000 00000111

^位异或运算(相同则0,不同则1)

eg : 3 ^ 4 = 7

数值 转换为二进制
3 00000000 00000000 00000000 00000011
4 00000000 00000000 00000000 00000100
^位运算后结果为7 00000000 00000000 00000000 00000111

^异或位运算符特点:
a = 10 , b = 20
(a ^ b ^ b) = 10
(a ^ b ^ a) = 20
一个数据对另一个数据位异或两次,该数本身不变

~按位取反运算符(0变1,1变0)

eg : ~3 = -4

数值 转换为二进制
3 00000000 00000000 00000000 00000011
~位运算后结果为-4 11111111 11111111 11111111 11111000(补码)
补码 11111111 11111111 11111111 11111000
反码 11111111 11111111 11111111 11111011
原码(-4) 10000000 00000000 00000000 00000100

字节大小

1 byte = 8 bit
1 k = 1024 byte
一个字节是八位
eg : 00100100(这是一个字节)

进制组成与换算

二进制

二进制(0b开头,由0,1组成)
eg : (10110110) = 2^7*1 + 2^6*0 + 2^5*1 + 2^4*1 + 2^3*0 + 2^2*1 + 2^1*1 + 2^0*0 = 182(十进制结果)

十进制

十进制(由0…..9组成)
eg : (182)

八进制

八进制(0开头,由0…7组成)
从右开始,每三位一个组合,最左边不够补0,然后分别算出十进制的值进行组合得到八进制的值.
eg : (0266) 010|110|110 = 2^2*0 + 2^1*1 + 2^0*0 | 2^2*1 + 2^1*1 + 2^0*0 | 2^2*1 + 2^1*1 + 2^0*0 = 266(八进制结果) = 2*8^2 + 6*8^1 + 6*8^0 = 182(十进制结果)

十六进制

十六进制(0x开头,由0….9,a…f组成)
同理与八进制,每四位一个组合.
eg : (0xB6) 1011|0110 = 2^3*1 + 2^2*0 + 2^1*1 + 2^0*1 | 2^3*0 + 2^2*1 + 2^1*1 + 2^0*0 = B6(十六进制结果) = 11*16^1 + 6*16^0 = 182(十进制结果)

原码,反码,补码

原码

原码 : 正数的原码最高位是0.负数的最高位是1,其他的是数值位.

值 符号位 数值位
+7 0 0000111
-7 1 0000111

反码

反码 : 正数的反码和原码相同.负数的反码与原码符号位相同,数值位取反(就是1变0,0变1).

值 符号位 数值位
+7 0 0000111
-7 1 1111000

补码

补码 : 正数的补码与原码相同.负数的补码是在反码的基础上加1.

值 符号位 数值位
+7 0 0000111
-7 1 1111001

CMD编译格式

javac + 文件名+ 扩展名 eg : javac HelloWorld.java (编译)
java + 类名(不加扩展名) eg : java HelloWorld (执行)

Java标识符及命名规则(见名知意)

标识符组成

数字、字母、$、_
注意 : Java关键字不可以作为标识符，(go to,const)为旧版本保留字段亦不可作为标识符使用。

包的命名

文件夹 单极 : xxx
多级 : xxx.xxx.xxx
(全部字母小写)

类或接口

一个单词 : 单词的首字母必须大写 eg : Student,Dog
多个单词 : 每个单词的首字母必须大写 eg : HelloWorld,StudentName

方法或变量

一个单词 : 单词的首字母小写 eg : age,student
多个单词 : 首字母小写,从第二个字母开始首字母大写 eg : studentAge,studentAllNames

常量

一个单词 : 全部大写 eg : PI
多个单词 : 每个字母都大写,用_隔开 eg : STUDENT_MAX_AGE
