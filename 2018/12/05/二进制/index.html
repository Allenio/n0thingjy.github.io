进制组成与换算

二进制

二进制(0b开头,由0,1组成)
eg : (10110110) = 2^7*1 + 2^6*0 + 2^5*1 + 2^4*1 + 2^3*0 + 2^2*1 + 2^1*1 + 2^0*0 = 182(十进制结果)

十进制

十进制(由0…..9组成)
eg : (182)

八进制

八进制(0开头,由0…7组成)
从右开始,每三位一个组合,最左边不够补0,然后分别算出十进制的值进行组合得到八进制的值.
eg : (0266) 010|110|110 = 2^2*0 + 2^1*1 + 2^0*0 | 2^2*1 + 2^1*1 + 2^0*0 | 2^2*1 + 2^1*1 + 2^0*0 = 266(八进制结果) = 2*8^2 + 6*8^1 + 6*8^0 = 182(十进制结果)

十六进制

十六进制(0x开头,由0….9,a…f组成)
同理与八进制,每四位一个组合.
eg : (0xB6) 1011|0110 = 2^3*1 + 2^2*0 + 2^1*1 + 2^0*1 | 2^3*0 + 2^2*1 + 2^1*1 + 2^0*0 = B6(十六进制结果) = 11*16^1 + 6*16^0 = 182(十进制结果)

原码,反码,补码

原码

原码 : 正数的原码最高位是0.负数的最高位是1,其他的是数值位.
eg : 值   符号位   数值位
      +7      0       0000111
      -7       1       0000111

反码

反码 : 正数的反码和原码相同.负数的反码与原码符号位相同,数值位取反(就是1变0,0变1).
eg : 值   符号位   数值位
      +7      0       0000111
      -7       1       1111000

补码

补码 : 正数的补码与原码相同.负数的补码是在反码的基础上加1.
eg : 值   符号位   数值位
      +7      0       0000111
      -7       1       1111001
