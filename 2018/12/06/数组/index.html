一维数组

定义格式:

A:数据类型[] 数组名;
B:数据类型 数组名[];
eg: A:int[] a;

推荐使用第一种.

二维数组

定义格式(标准):

数据类型[ ][ ] 数组名 = new 数据类型[m][n];
m表示这个二维数组有多少个一维数组.
n表示每一个一维数组的元素个数.

其他格式:

a:数据类型 数组名[ ][ ] = new 数据类型[m][n];
b:数据类型[ ] 数组名[ ] = new 数据类型[m][n];
c:数据类型[ ][ ] 数组名 = new 数据类型[m][n];(标准)
d:数据类型[ ][ ] 数组名 = new 数据类型[m][ ];(一维数组元素个数可以动态给出)
e:数据类型[ ][ ] 数组名 = new 数据类型[ ][ ]{{元素...},{元素...},{元素...}};(直接赋值)
f:数据类型[ ][ ] 数组名 = {{元素...},{元素...},{元素...}};(e的简化版)

传递值问题

基本类型传递值   --->   值(不改变方法中调用的值)
引用类型传递值   --->   地址值(改变方法中调用的值)
